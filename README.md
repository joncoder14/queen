# Queen
 Queen is a British rock band formed in London in 1970. The group became famous for its unique sound, combining rock, opera, and pop styles. Its most iconic members were Freddie Mercury (lead singer),
 Brian May (guitarist), Roger Taylor (drummer), and John Deacon (bassist).

They are known for legendary songs like Bohemian Rhapsody, We Will Rock You, and We Are the Champions. Queen is considered one of the greatest rock bands of all time.

## technologies used:
1. HTML5
2. CSS3

## Project structure
The project follows a simple semantic HTML structure:

 - `<header>` → Title, main heading and navigation menu.
 - `<main>` → Main content.
 - `<footer>` → social media, biographies of the members and Copyright.

## Run the project locally

This is a static web project built with HTML, CSS and JavaScript, so no additional dependencies or installations are required.

1. Clone the repository
git clone https://github.com/joncoder14/queen.git
cd queen
2. Open the project

You can run the project in two simple ways:

Option A: Open directly in your browser
Locate the index.html file
Double click it

or

Right click → Open with → Your preferred browser
Option B: Use Live Server (recommended)

For a better development experience:

Open the project in VS Code
Install the Live Server extension
Right click on index.html
Select "Open with Live Server"

The project will run on a local server like:

http://localhost:5500
